    # vue中的动画

    元素的显示隐藏都可以增加动画效果v-if、v-show、v-for、路由切换等操作。
    常见的增加动画的方式有 animation 、 transition 、 js编写动画

    今天主要看看vue内置的css动画：

    从图片可以看出，css常用的类名有以下6个：

    • .v-enter： 触发enter之前
    • .v-enter-active： 进入动画过程
    • .v-enter-to： 进入动画结束
    • .v-leave： 触发leave之前
    • .v-leave-active： 离开动画过程
    • .v-leave-to： 离开动画结束

    基本的实例：

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
    <style >
        .content{
            width: 100px;height: 100px;background: red;
        }
        .v-enter{ /*进入前， 这里会导致默认的红色一闪而逝，基本看不见*/
            background: blue;
        }
        .v-enter-active{
            background: purple;
            transition: all 10s linear;
        }
        .v-enter-to{
            background: chartreuse;  /* 结束之后，变成默认的红色 */
        }
        
        /* 离开过程 */
        .v-leave{       /* 这一步的颜色也基本看不见，直接有默认的红色 => 进入到 .v-leave-active */
            background: rgb(98, 0, 255);
        }
        .v-leave-active{
            background: crimson;
            transition: all 10s linear;
        }
        .v-leave-to{
            background: yellow;
        }   
        /*最终 消失 */
    </style>
</head>

<body>
    <!-- VUE 中的动画 v-if v-for v-show 路由切换 -->
    <script src="node_modules/vue/dist/vue.min.js"></script>
    <div id="app">
        <button @click="change">点击我</button>
        <transition>
            <div v-if="isShow" class="content">
                content
            </div>
        </transition>
    </div>
    <script>
        let vm = new Vue({
            el: '#app',
            data: {
                isShow: false
            },
            methods: {
                change() {
                    this.isShow = !this.isShow
                }
            }
        })
    </script>
</body>

</html>
