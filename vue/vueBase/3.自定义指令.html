    # 自定义指令

    自定义指令分为全局指令和局部指令

    定义指令前，需要清楚指令有哪几部分组成：eg: v-model.a.b.c = 'value'

    • 指令名：model（v- 是自动添加的）
    • 修饰符：a.b.c 可以有多个修饰符一起用
    • 绑定的值：value，一般都是data中定义的变量

    # 一、自定义指令的生命周期钩子

    bind -> inserted -> update -> componentUpdated -> unbind

    • bind：只调用一次，指令第一次绑定到元素时调用。在这里可以进行一次性的初始化设置。
    • inserted：被绑定元素插入父节点时调用 (仅保证父节点存在，但不一定已被插入文档中)。
    • update：所在组件的 VNode 更新时调用，但是可能发生在其子 VNode 更新之前。
    • componentUpdated：指令所在组件的 VNode 及其子 VNode 全部更新后调用。
    • unbind：只调用一次，指令与元素解绑时调用。

    总结：

    • bind + update 这两个钩子相当于 inserted
    • 所以，实际运用中我们常用 inserted、unbind 即可

    # 二、钩子函数参数

    钩子函数参数有四个：el、binding、vnode 和 oldVnode

    • el: 指令所绑定的元素，可以用来直接操作 DOM。
    • bindings: 一个对象，包括自定义指令中的各个属性, 属性截图如下：
    • vnode: Vue 编译生成的虚拟节点
    • oldnode: 上一个虚拟节点， 仅在 update 和 componentUpdated 钩子中可用。

    # 三、全局指令

    <div id="app">
    <input type="text" v-focus.color="color">
</div>
<script src="node_modules/vue/dist/vue.js"></script>
<script>
    // =======>>>  注意：自定义指令需要定义在实例化Vue之前，否则将报错
    Vue.directive('focus',{
        inserted(el,bindings,vnode,oldnode){ // 指令元素插入到页面时执行
            /*
                el: 指令所绑定的元素，可以用来直接操作 DOM。
                bindings: 一个对象，包括自定义指令中的各个属性
                vnode: Vue 编译生成的虚拟节点
                oldnode: 上一个虚拟节点
            */
            if(bindings.modifiers.color){       // 如果有修饰符color，怎改变dom放入背景颜色
                el.style.background = bindings.value
            }
            el.focus();
        },
        unbind(){ // 指令销毁的时候 会触发此函数

        }
    })
        
    let vm = new Vue({
        el:'#app' , // querySelector
        data(){
            // 这里就是model层
            return {
                color:'red'
            };
        }
    })
</script>

    # 四、局部指令

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
    <style>
        input {
            border: 1px solid red
        }
        .content {
            width: 100px;
            height: 100px;
            background: burlywood;
        }
    </style>
</head>

<body>
    <div id="app">
        <!-- 日历 弹层 -->
        <div v-click-outside>
            <input type="text">
            <div class="content" v-if="isShow">content
                <button>点我 </button>
            </div>
        </div>

    </div>
    <script src="node_modules/vue/dist/vue.js"></script>
    <script>
        // 可以实现 点击时判断是否存在当前的dom中
        let vm = new Vue({
            el: '#app',
            directives: { // 可以声明多个所以是复数形式
                clickOutside: {
                    bind(el, bindings, vnode) {
                        el.fn = (e) => {
                            if (el.contains(e.target)) {  // 事件源(被点击的dom)是否包含el中
                                vnode.context['focus']()
                            } else {
                                vnode.context['blur']()
                            }
                        }
                        document.addEventListener('click', el.fn)     // 用事件代理，定义click事件和出来函数
                    },
                    unbind(el) { // 事件绑定必须要解绑
                        document.removeEventListener('click', el.fn)        // 移除事件监听
                    }
                }
            },
            data: {
                isShow: false
            },
            methods: {
                blur() {
                    this.isShow = false;
                },
                focus() {
                    this.isShow = true;
                }
            }
        });
    </script>
</body>
</html>
