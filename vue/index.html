    # 该专栏主要包括 vue基础、vue-router、vuex、vue其他 几方面的知识!

    # Vue核心概念及特性

    Vue (读音 /vjuː/，类似于 view) 是一套用于构建用户界面的渐进式 框架（react给自己的定义是：用于构建用户界面的 JavaScript 库）。
    特点: 易用，灵活，高效 渐进式框架

    可以随意组合需要用到的模块 vue + components + vue-router + vuex + vue-cli

    # 1. 为什么称为渐进式？

    • 因为vue全家桶还是一个比较庞大的技术体系，不似于Jquery，学完之后就可以依靠它做项目！而vue中想要做一个比较完整的项目，需要用到的模块还是比较多的。但vue之所以被称作渐进式，是因为这些模块并没有内置在vue框架中，而是以第三方模块单独存在，用户想要用起来vue并不需要学完所有的模块，而是可以根据具体的项目要求，引入自己需要的模块！！！

    # 2. 什么是库，什么是框架?

    • 库是将代码集合成一个产品,库是我们调用库中的方法实现自己的功能。 如Jquery: $.html()
    • 框架则是为解决一类问题而开发的产品,框架是我们在指定的位置编写好代码，框架帮我们调用。 如Vue: 将方法写在methods中，vue将在对应的时机去调用

    框架与库之间最本质区别在于控制权：you call libs, frameworks call you（你调用的是库，而框架是调用你写的代码）

    Vue属于框架

    # 2.MVC模型 && MVVM模型


    现在的MVVM，其实是将传统的MVC中的V(view)层拆分出来的！
    Vue是MVVM，而React其实更多的是单独的V(view，react是构建页面的)，但很多时候都将react称为MVC

    在传统的mvc中除了model和view以外的逻辑都放在了controller中，导致controller逻辑复杂难以维护,在mvvm中view和model没有直接的关系，全部通过viewModel进行交互
