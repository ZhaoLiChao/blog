遍历(一)

这里总结的Array遍历方法有一个特点，callBack函数会为数组中的每个索引调用，即从0到arr.length-1，而不仅仅是那些被赋值的索引，这意味着对于稀疏数组来说，该方法的效率要低于那些只遍历有值的索引的方法。

# Array.prototype.find()

find()方法返回数组中第一个满足测试函数(callback)的元素的值(区别于findIndex返回元素的索引)，否则返回undefined

arr.find(callBack[, thisArg])
  • callBack：用于测试每个元素的函数，参数为(element, index, arr)
  • thisArg：执行callBack时使用的this值

描述：find()方法为数组中的每个元素执行一次callBack函数，直到找到一个使callBack返回true的元素。如果找到这个元素，那么find()方法立即返回这个元素的值；否则返回undefined；注意find()方法会遍历数组中的每一个元素，包括稀疏数组的empty slots！

DEMO

function isAnimal(item, index, arr){
    return item.isAnimal;
}

var arr = [
    {name: 'Apple', isAnimal: false},
    {name: 'Dog', isAnimal: true},
    {name: 'Banana', isAnimal: false}
]
var arr2 = [
    {name: 'Apple', isAnimal: false},
    {name: 'Orange', isAnimal: false},
    {name: 'Banana', isAnimal: false}
]

arr.find(isAnimal);//{name: 'Dog', isAnimal: true}
arr2.find(isAnimal);//undefined

# Array.prototype.findIndex()

findIndex()方法返回数组中第一个满足测试函数(callback)的元素的索引(区别于find返回元素的值)，否则返回-1

arr.findIndex(callBack[, thisArg])
  • callBack：用于测试每个元素的函数，参数为(element, index, arr)
  • thisArg：执行callBack时使用的this值

描述：findIndex()方法为数组中的每个元素执行一次callBack函数，直到找到一个使callBack返回true的元素。如果找到这个元素，那么find()方法立即返回这个元素的索引；否则返回-1；注意findIndex()方法会遍历数组中的每一个元素，包括稀疏数组的empty slots！

DEMO

function isAnimal(item, index, arr){
    return item.isAnimal;
}

var arr = [
    {name: 'Apple', isAnimal: false},
    {name: 'Dog', isAnimal: true},
    {name: 'Banana', isAnimal: false}
]
var arr2 = [
    {name: 'Apple', isAnimal: false},
    {name: 'Orange', isAnimal: false},
    {name: 'Banana', isAnimal: false}
]

arr.findIndex(isAnimal);//1
arr2.findIndex(isAnimal);//-1

遍历(二)

这篇笔记总结的Array遍历方法和上一篇相反，callBack函数只会为数组中被赋值的索引调用。那些已删除或者未初始化的项将被跳过(稀疏数组)。这些方法的callBack函数在第一次调用时就已经确定了范围。新增加的元素将不会被callBack访问到；若已经存在的元素被改变或者删除，则它们传递到callBack的值是遍历到它们的那一时刻的值，被删除的元素将不会被访问到。个人理解是在遍历开始时，索引已经确定，之后每次遍历无论arr怎么变化，只会根据当前索引取当前arr的元素。

# Array.prototype.forEach()

forEach()方法对数组的每个元素执行一次提供的函数

arr.forEach(callBack[, thisArg])
  • callBack：用于测试每个元素的函数，参数为(element, index, arr)
  • thisArg：执行callBack时使用的this值

描述：forEach()方法为数组中含有效值的每一项执行一次callBack函数。

DEMO

var arr = [1, , 2, 3];

//跳过第二个索引处
arr.forEach((v, i) => {
    console.log('第'+ (i + 1) + '个元素值为' + v);
    //第1个元素值为1
    //第3个元素值为2
    //第4个元素值为3
});

//数组迭代时被修改，由于遍历已经确定范围，即索引不变
var numbers = [1, 2, 3, 4];
numbers.forEach((v, i) => {
    console.log(v,i);
    if(3 === v){
        numbers.shift();
        numbers.push(5);
    }
});
//1 0
//2 1
//3 2
//5 3

# Array.prototype.every()

every()方法用于测试数组的所有元素是否都通过了指定函数的测试

arr.every(callBack[, thisArg])
  • callBack：用于测试每个元素的函数，参数为(element, index, arr)
  • thisArg：执行callBack时使用的this值

描述：every()方法为数组中的每个元素执行一次callBack函数，直到找到一个使callBack返回false的元素。如果找到这个元素(返回值为false)，那么every()方法立即返回false,且不在执行之后的遍历；否则every()方法返回true。

DEMO

function isPositive(item, index, arr){
    return item > 0;
}

[1, -1, 3, 0].every(isPositive);//false
[7, 21, 5, 0.6].every(isPositive);//true

# Array.prototype.some()

some()方法用于测试数组的某些元素是否都通过了指定函数的测试

arr.some(callBack[, thisArg])
  • callBack：用于测试每个元素的函数，参数为(element, index, arr)
  • thisArg：执行callBack时使用的this值

描述：some()方法为数组中的每个元素执行一次callBack函数，直到找到一个使callBack返回true的元素。如果找到这个元素(返回值为true)，那么some()方法立即返回true,且不在执行之后的遍历；否则some()方法返回false。

DEMO

function isPositive(item, index, arr){
    return item > 0;
}

[1, -1, 3, 0].some(isPositive);//true
[-7, 0, -5, -8].some(isPositive);//false
[7, 21, 5, 0.6].some(isPositive);//true

# Array.prototype.filter()

filter()方法创建了一个新数组，包含所有通过callBack函数测试的元素

var newArray = oldArray.filter(callBack[, thisArg])
  • callBack：用来测试数组的每个元素的函数，参数为(element, index, arr)
  • thisArg：执行callBack时使用的this值

描述：filter()方法为数组中的每个元素执行一次callBack函数，并将所有callBack返回true的元素创建一个新数组。

DEMO

function positive(item, index, arr){
    return item > 0;
}

[-21, -1, -3, 0].filter(positive);//[]
[1, -1, 3, 0].filter(positive);//[1, 3]
[7, 21, 5, 0.6].filter(positive);//[7, 21, 5, 0.6]

# Array.prototype.map()

map()方法创建了一个新数组，其结果是该数组中的每个元素调用一个提供的函数后返回的结果。

var newArray = oldArray.map(callBack[, thisArg])
  • callBack：用来测试数组的每个元素的函数，参数为(element, index, arr)
  • thisArg：执行callBack时使用的this值

描述：map()方法为数组中的每个元素执行一次callBack函数，并将所有callBack的返回值(包括undefined)创建一个新数组。map()方法在处理数组时，数组元素的范围在callBack函数第一次调用就已经确定。被删除的元素不会被访问，但是会返回undefined(empty slots)。

DEMO

var numbers = [1, 4, 9];
numbers.map(v => v * 2);
//[2, 8, 18]

var words = ['a', , 'c', 'd'];
words.map(v => {
    console.log(v);
    return v;
});
//a
//注意第二个索引出没有被遍历
//c
//d
//虽然没有参与遍历，但是输出时以undefined(empty slots)作为结果
//['a', undefined ×1, 'c', 'd']

//字符反转
var str = 'Hello';
Array.prototype.map.call(str,function(v){
    return v;
}).reverse().join('');
//olleH

# Array.prototype.reduce()

reduce()方法对累加器和数组中的每个元素(从左到右)应用一个函数，将其减少为单个值

arr.reduce(callBack[, initialValue])
  • callBack：执行数组中每个值的函数，包含四个参数
    • accumulator：累加器累加回调的返回值；可以是上次回调后返回的累加值，或者是initialValue初始值。
    • currentValue：当前遍历元素的值
    • currentIndex：当前遍历元素的索引。如果提供了initialValue，则索引为0，否则从1开始计算。
    • array：当前数组
  • initialValue：初始值，如果没有提供初始值，那么以第一个元素作为初始值。对于没有提供初始值的空数组，将抛出TypeError。提供初始值通常更安全。

DEMO

[0, 1, 2, 3, 4].reduce((prev, curr) => prev + curr,5);
//15

[0, 1, 2, 3, 4].reduce((prev, curr) => prev - curr,5);
//-5

//二维数组转化为一维
[[0, 1], [2, 3], [4, 5]].reduce((prev, curr) => {
    return prev.concat(curr);
    //return [...prev, ...curr];
},[]);
//[0, 1, 2, 3, 4, 5]

# Array.prototype.reduceRight()

reduceRight()方法对累加器和数组中的每个元素(从右到左)应用一个函数，将其减少为单个值

arr.reduceRight(callBack[, initialValue])
  • callBack：执行数组中每个值的函数，包含四个参数
    • accumulator：累加器累加回调的返回值；可以是上次回调后返回的累加值，或者是initialValue初始值。
    • currentValue：当前遍历元素的值
    • currentIndex：当前遍历元素的索引。如果提供了initialValue，则索引为最后一个元素的索引，否则从倒数第二个元素的索引开始计算。
    • array：当前数组
  • initialValue：初始值，如果没有提供初始值，那么以最后一个元素作为初始值。对于没有提供初始值的空数组，将抛出TypeError。提供初始值通常更安全。

DEMO

[0, 1, 2, 3, 4].reduceRight((prev, curr) => prev + curr,5);
//15

[0, 1, 2, 3, 4].reduceRight((prev, curr) => prev - curr,5);
//-5

//二维数组转化为一维
[[0, 1], [2, 3], [4, 5]].reduceRight((prev, curr) => {
    return prev.concat(curr);
    //return [...prev, ...curr];
},[]);
//[4, 5, 2, 3, 0, 1]
