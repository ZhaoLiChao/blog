总结三个判断元素是否存在数组中的方法

# Array.prototype.indexOf()

indexOf()方法返回在数组中找到给定元素的第一个索引，如果不存在，则返回-1

arr.indexOf(searchValue[, fromIndex])
  • searchValue：要查找的元素
  • fromIndex：开始查找的位置，默认为0；如果该索引值大于或等于arr.length，意味着不会从数组中查找，返回-1。如果索引值为负数，自动计算为arr.length+fromIndex。

描述：indexOf()方法使用的是===比较运算符进行匹配，即匹配时会必须完全匹配。

DEMO

var array = [2, 5, 9];

array.indexOf(2);     
//0
array.indexOf(7);     
//-1
array.indexOf(9, 2);  
//2

//获取arr中所有指定元素的位置
var array = ['a', 'b', 'a', 'a', 'd'];
var element = 'a';
var indices = [];

var index = array.indexOf(element);
while(-1 !== index){
    indices.push(index);
    index = array.indexOf(element,++index);
}

# Array.prototype.lastIndexOf()

lastIndexOf()方法返回在数组中找到给定元素的最后一个索引，如果不存在，则返回-1

arr.lastIndexOf(searchValue[, fromIndex])
  • searchValue：要查找的元素
  • fromIndex：开始逆向查找的位置，默认为arr.length-1；如果该索引值大于或等于arr.length，意味着整个数组都将被查找。如果索引值为负数，自动计算为arr.length+fromIndex。

描述：lastIndexOf()方法使用的是===比较运算符进行匹配，即匹配时会必须完全匹配。

DEMO

var array = [1, 2, 3, 4, 5];

array.lastIndexOf(2);
//1
array.lastIndexOf(7);
//-1
array.lastIndexOf(2, 4);
//1,从第四个索引出开始向前查找
array.lastIndexOf(2, -3);
//1,-3自动计算为2(array.length+(-3))

# Array.prototype.includes()

includes()方法用来判断一个数组是否包含按一个指定的值，如果包含返回true，否则返回false。

arr.includes(searchValue[, fromIndex])
  • searchValue：要查找的元素
  • fromIndex：开始查找的位置，默认为0；如果该索引值大于或等于arr.length，意味着不会从数组中查找，返回-1。如果索引值为负数，自动计算为arr.length+fromIndex。

描述：indexOf()方法使用的是===比较运算符进行匹配，即匹配时会必须完全匹配。相较于indexOf()方法，includes()可以判断NaN。

DEMO

var array = [2, 5, 9, 'a'];

array.indexOf(2);     
//true
array.indexOf(2, 3);     
//false
array.indexOf(7);     
//false
array.indexOf('a');  
//true
